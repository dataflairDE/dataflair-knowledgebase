## Sie möchten Java 21 auf Ihren Server installieren?


Server-Paketliste aktualisieren
``` bash
apt update && apt upgrade -y
```

Java 21 installieren
``` bash
apt install openjdk-21-jre-headless -y
```

Überprüfung der Java-Version
``` bash
java -version
```
